# scripts/dev/

Utilitários de desenvolvimento e diagnóstico. Não fazem parte do pipeline oficial.

Scripts aqui são temporários ou de apoio: verificação de dados, debug, testes manuais.

## Convenção
- `check_*.py` — verificação de estado da BD ou dados
- `debug_*.py` — debugging pontual
- `seed_*.py` — dados de teste locais

## Pipeline oficial
O script de produção está em `scripts/import_gpx.py`.
